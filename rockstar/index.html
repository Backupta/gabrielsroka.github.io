rockstar bookmarklet

Enhance Okta with these features:
  • Export Objects to CSV: eg, Users, Groups, Directory Users, App Users, App Groups, Apps, ...
  • Administrators page: Export Admins
  • User home page: Show SSO (SAML assertion, etc)
  • People page: enhanced search, Export Users menu
  • Person page: show login/email and AD info, show user detail, enhance menus/title, manage user's admin roles
  • Events: Expand All and Expand Each Row
  • API: Pretty Print JSON
  • SU Orgs & Org Users: enhanced search
  • Many: enhanced menus
...and more to come

Setup

  1. Show your bookmarks toolbar. In Chrome, ... > Bookmarks > Show Bookmarks Bar. In Firefox, right-click in the title bar and click Bookmarks Toolbar.
  2. Drag this to the bookmarks toolbar:
    rockstar
Mac Users, if Chrome doesn't work using the instructions above, try:
  1. Right-click on the rockstar "button" above and Copy Link Address
  2. Bookmark this page, Edit the bookmark, Tab to URL, Paste

Usage

  1. Login to Okta or Okta Admin.
  2. Navigate to, e.g., Directory > People or Security > Administrators, etc.
  3. Click the rockstar bookmark from your toolbar.
  4. Look for the rockstar popup window in the upper-left corner of your browser.
